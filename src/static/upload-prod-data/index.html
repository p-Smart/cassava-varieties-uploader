Usage:

            

                
For JSON Array

                
An array of objects with the keys below is expected:


{
    "Title": "string",
    "Country": "string",
    "Name": "string",
    "Organization": "string",
    "Address": "string",
    "Location": "string",
    "Variety": "string",
    "Variety Details": "string",
    "Type of Seed": "string",
    "Date of planting": "string",
    "Size of Field": "string",
    "Production Year": "string",
    "Phone": "string",
    "Email": "string"
}


            

    
            

                
For Excel File,

                
Rename the sheet containing the data to be uploaded to data.

                The first row must contain the following headings: "Title", "Country", "Name", "Organization", "Address", "Location", "Variety", "Variety Details", "Type of Seed", "Date of planting", "Size of Field", "Production Year", "Phone", and "Email"
                

            

    
            
All keys are case-sensitive.
